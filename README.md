# E-commerce Admin Demo

Небольшой демонстрационный проект административной панели интернет-магазина на React, TypeScript и Vite.

## Что реализовано

- Адаптивная административная панель с боковым меню
- Дашборд с ключевыми метриками и активностью
- Страница товаров с формой добавления и таблицей остатков
- Страница заказов с визуализацией статусов
- Страница клиентов с карточками сегментов
- Современный UI на Tailwind CSS
- Работа с асинхронными данными через React Query
- Формы с валидацией и управлением состоянием через React Hook Form

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- React Query
- React Hook Form
- Lucide React

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
